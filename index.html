Gruppe 2:

Routenplanung zum Standort Heilbronn: Eingabe eines Startpunktes der Route im Aktionsbereich und Darstellung der Route zum Standort Heilbronn in Google Maps. Kartenansicht „Roadmap“

Aufgabe 1:

Allgemein Die Benutzeroberfläche ist mittels Responsive Design für mobile Endgeräte, Tablets und PC zu optimieren. Die Verwendung eines Frameworks zur Umsetzung (Bootstrap, Foundation o.ä.) wird empfohlen. Für die Umsetzung wird die Integration von Google Maps benötigt: https://developers.google.com/maps/visualize?hl=de Teilweise können Lösungen die Verwendung eines API Keys (Google Developer Programm) verlangen. Ist dies der Fall, ist der Weg zur Erzeugung eines API Keys im Google Developer Portal beschrieben. Zur Erzeugung eines API Keys benötigen Sie einen Google Account. Diesen können Sie kostenlos bei Google Erzeugen.
